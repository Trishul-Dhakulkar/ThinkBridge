*Please follow the following steps to run the Shop_Bridge Application

1)Set Connection String in Web.Config File to Connect to the database.

2)Run the Migrations to update database.

3)Run the application and use OpenAPI Swagger Tool for Testing the Web API
  - follow the steps provided in Screenshots under Documentation folder 
    which will guide you how to run application and test the API using Swagger-UI
  - Sample Json file is available to Test AddProduct API Call in-case of passing
    Imagedata you need to pass Base64 encoded string to ImageData field to save Product Image.
